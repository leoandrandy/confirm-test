# お問い合せフォーム

## 環境構築
- git cloneにてリポジトリを設定
- docker-compose up -d -build
- docker-compose exec php bashでPHPコンテナに入る
- composer installでパッケージをインストール
- .env,exampleをコピーして.envを作成し、DBの内容をdocker-compose.ymlファイルにあわせて変更

## 使用技術(実行環境)
- laravel Framework 8.83.8
- PHP 7.4.9
- phpMyAdmin 5.2.1
